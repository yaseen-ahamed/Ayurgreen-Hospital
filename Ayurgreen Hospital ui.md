# AYURGREEN WEBSITE UI SYSTEM (ANTIGRAVITY)

---

## 1. CORE DESIGN DIRECTION

- Minimal, premium, calm
- Healthcare + nature fusion
- Soft, breathable layout
- No visual noise
- Motion > decoration

---

## 2. COLOR SYSTEM

### Primary Colors
- Deep Green: #0F3D2E
- Forest Green: #1F5A44
- Soft Green Tint: #EAF4F0

### Neutral Base
- Background: #F7F8F7
- Card Background: #FFFFFF
- Border: #E5E7EB

### Text
- Primary: #111111
- Secondary: #6B7280
- Muted: #9CA3AF

### Accent
- Subtle Gradient:
  - Green → Transparent overlay
- Hover Accent: slightly brighter green

---

## 3. TYPOGRAPHY

### Font Style
- Clean, modern sans-serif
- No decorative fonts

### Hierarchy

#### H1 (Hero)
- Large, bold
- Tight line height
- Slight negative letter spacing

#### H2 (Section Headings)
- Medium-large
- Clean spacing
- Balanced weight

#### H3 (Sub-sections)
- Slightly bold
- Compact

#### Body
- Comfortable reading size
- Medium line height

#### Micro Text
- Small
- Muted color

---

## 4. LAYOUT SYSTEM

### Container
- Max width: 1200–1320px
- Center aligned

### Section Spacing
- Large vertical spacing
- 80–120px padding top/bottom

### Grid
- 12 column structure
- Generous gaps

---

## 5. BORDER RADIUS

- Primary radius: 16px
- Large elements: 24px
- Pills / buttons: full rounded

---

## 6. SHADOW SYSTEM

### Default
- Very soft shadow
- Almost invisible

### Hover
- Slight elevation
- Subtle blur shadow

---

## 7. HEADER / NAVBAR

### Style
- Floating
- Slight glass / translucent feel
- Rounded edges

### Nav Items
- Capsule buttons
- Light border
- Hover: soft fill

### CTA Button
- Solid dark (black or deep green)
- Rounded pill
- Icon inside

---

## 8. HERO SECTION

### Layout
- Full-width visual
- Rounded container edges

### Overlay
- Gradient overlay (dark → transparent)

### Content Position
- Left aligned
- Vertical center

### CTA
- Primary button (filled)
- Icon inside circle

---

## 9. CARD SYSTEM

### Structure
- Rounded corners
- Image + overlay text

### Hover Behavior
- Slight scale (1.02)
- Shadow increase

### Overlay
- Gradient fade for readability

## 10. BUTTON SYSTEM

### Primary Button
- Filled (black / deep green)

### Secondary Button
- Outline style
- Light border

### Hover
- Slight scale
- Background shift

## 11. ICON SYSTEM

### Style

- Line or solid monochrome

- No flashy effects

## 12. CAROUSEL / SLIDER

- Circular arrow buttons

## 13. IMAGE STYLE

- Soft contrast

- Rounded (consistent system)
